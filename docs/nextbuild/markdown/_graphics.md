# Graphics 

Graphics can be broken down into cetgories:

| Graphics Type | Description |
|---------------|-------------|
| [Layer 2](_layer2.md) | 256x192, 320x256, 640x256 resolutions |
| [Sprites](_sprites.md) | Hardware-accelerated sprite system |
| [TileMaps](_tilemaps.md) | Tile-based graphics system |
| [Palettes](_palettes.md) | Color palette management |

Other systems that can affect Graphics :

| |  |
|---------------|:-------------------------|
| [DMA](_DMA.md) | Direct Memory Access for fast data transfers |
| [COPPER](_COPPER.md) | Hardware-accelerated display list processor |
