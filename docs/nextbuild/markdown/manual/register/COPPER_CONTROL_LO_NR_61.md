# COPPER_CONTROL_LO_NR_61

## Syntax

```
COPPER_CONTROL_LO_NR_61 = $61
```

## Description

**COPPER CONTROL LOW BYTE** (R/W)

More info : https://wiki.specnext.dev/Copper_Control_Low_Byte

Holds low byte of Copper control bits.

Low eight bits of [Copper](https://wiki.specnext.dev/Copper) list index.Index is set to 0 after a reset.
