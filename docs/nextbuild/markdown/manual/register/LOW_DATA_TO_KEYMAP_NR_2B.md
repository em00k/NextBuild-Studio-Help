# LOW_DATA_TO_KEYMAP_NR_2B

## Syntax

```
LOW_DATA_TO_KEYMAP_NR_2B = $2B
```

## Description

**KEYMAP LOW DATA REGISTER**

More info : https://wiki.specnext.dev/Keymap_Low_Data_Register

Low eight LSBs of PS/2 Keymap data.

By writing to this register the keymap address is auto-incremented.
