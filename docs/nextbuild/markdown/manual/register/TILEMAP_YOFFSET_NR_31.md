# TILEMAP_YOFFSET_NR_31

## Syntax

```
TILEMAP_YOFFSET_NR_31 = $31
```

## Description

**TILEMAP OFFSET Y REGISTER**

More info : https://wiki.specnext.dev/Tilemap_Offset_Y_Register

Sets the pixel offset used for drawing Tilemap graphics on the screen.

Tilemap Y-offset (0-255) (0 after a reset)

	#INCLUDE <nextlib.bas>
