# MMU0_0000_NR_50

## Syntax

```
NextReg(MMU0_0000_NR_50,n ) 
```

## Description

**MEMORY MANAGEMENT SLOT 0 BANK** (R/W)

n = memory bank 0-224 on a fully expanded Next

Selects the 8k-bank stored in 8k-slot 0 [see Memory map](https://wiki.specnext.dev/Memory_map).
```
Slot 0 : $0000-$1FFF
```
