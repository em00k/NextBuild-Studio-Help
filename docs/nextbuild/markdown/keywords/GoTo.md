# GOTO

## Syntax

```
GOTO *label*
```

## Description

Jump to GO TO label

**Example**

	IF A = 10 GOTO APPLES
	IF A = 0 GOTO NOAPPLE
	APPLES:
		PRINT "WE HAVE APPLES" : STOP
	NOAPPLES:
		PRINT "NO APPLES"

**Remarks**
- This is compatible with Sinclair Basic

More info : https://zxbasic.readthedocs.io/en/docs/
