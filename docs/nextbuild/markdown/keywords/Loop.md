# LOOP

## Syntax

```
DO ... LOOP
```

## Description

DO ... LOOP Seen DO for more information 

Infinite loops

	DO
	sentences
	LOOP: This loop repeats forever.

Looping UNTIL

	DO
	sentences
	LOOP UNTIL condition

You can also put the condition at the beginning, this way:

	DO UNTIL condition
	sentences
	LOOP
