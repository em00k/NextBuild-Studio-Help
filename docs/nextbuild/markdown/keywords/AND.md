# AND

## Syntax

```
a AND b
```

## Description

AND is a Logical Operator. Performs the Logical Conjunction and returns TRUE if and only if both arguments are TRUE. 

**Example**

	a = b AND c

	| A | B | Result |
	| F | F | 	F	 |
	| F | T | 	F	 |

	| T | T | 	T	 |
