# XOR

## Syntax

```
a XOR b
```

## Description

XOR is a Logical Operator. Performs a logical XOR and returns TRUE if one of the arguments is true and one of the arguments is false. In essence, returns true if ONLY one of the arguments is true. 

**Example**

	a = b XOR c

	| A | B | Result |
	| F | F | 	F	 |
	| F | T | 	T	 |
	| T | F | 	T	 |
	| T | T | 	F	 |
