# BOLD

## Syntax

```
BOLD *n* or PRINT BOLD *n*;
```

## Description

Where n is either 0 (false) or 1 (true), then the subsequent print statements will have their INK pixels emphasized, making text appear bolder.

**Example**

	Print INK 0;PAPER 7; BOLD 1; "This is BOLD BLACK text on WHITE"

Remarks

- This statement is NOT Sinclair BASIC compatible

More info : https://zxbasic.readthedocs.io/en/docs/bold/
