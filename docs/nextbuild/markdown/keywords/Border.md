# BORDER

## Syntax

```
BORDER n ubyte
```

## Description

This can be used to change the border colour. Where n is a number  between 0 and 7.

**Example**

	BORDER 2
- 0 - Black
- 1 - Blue
- 2 - Red
- 3 - Magenta
- 4 - Green
- 5 - Cyan
- 6 - Yellow
- 7 - White

Remarks

- This statement is Sinclair BASIC compatible

More info : https://zxbasic.readthedocs.io/en/docs/border/
