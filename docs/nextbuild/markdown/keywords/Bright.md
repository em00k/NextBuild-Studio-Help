# BRIGHT

## Syntax

```
BRIGHT *n* or PRINT BRIGHT *n*;
```

## Description

Where n is either 0 (false) or 1 (true), where n is either 0 (false) or 1 (true), then the subsequent print statements will have both their INK and PAPER values set to the higher intensity BRIGHT mode.

**Example**

	Print INK 0;PAPER 7; BRIGHT 1; "This is BLACK text on BRIGHT WHITE background"

Remarks

- This statement is Sinclair BASIC compatible

More info : https://zxbasic.readthedocs.io/en/docs/bright/
