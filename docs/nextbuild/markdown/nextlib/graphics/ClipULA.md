# CLIPULA

## Syntax

```
ClipULA(x1, x2, y1, y2)
```

## Description

Sets the visible clip area of ULA layer. 

**Example**

Clip ULA window to maximum visible size :

```
ClipULA(0,255,0,255)
```

**Remarks**

This command is a shortcut configuring [CLIP_ULA_LORES_NR_1A](CLIP_ULA_LORES_NR_1A.md) 4 times.
