# CLIPSPRITE(x1, x2, y1, y2)

## Syntax

```
ClipSprite(x1, x2, y1, y2)
```

## Description

Sets the visible clip area of Layer2.

**Example**

Clip Sprites window to maximum visible size :

```
ClipSprite(0,255,0,255)
```
**Remarks**

This command is a shortcut configuring [CLIP_SPRITE_NR_19](CLIP_SPRITE_NR_19) 4 times. 

Requires: 

```
#INCLUDE <nextlib.bas>
```
