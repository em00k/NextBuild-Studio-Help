# CLS320

## Syntax

```
CLS320(n ubyte)
```

## Description

Clears Layer2 320x256 mode with colour n. This will clear the entire layer to the colour n.

**Examples**
```
CLS320(0) ' Clear Layer2 320x256 mode to black
```
## Notes

It is preferable to use InitLayer2 and ClearLayer2 to clear the layer instead, as this will set up the layer2 subsystem and clear the layer to the correct colour. See [InitLayer2](INITLAYER2.md) and [ClearLayer2(nColour)](CLEARLAYER2.md) for more information.
