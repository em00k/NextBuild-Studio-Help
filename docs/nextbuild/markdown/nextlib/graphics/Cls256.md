# CLS256 {deprecated}

## Syntax

```
CLS256(n ubyte)
```

## Description

Clears Layer2 256x192 mode with colour n. This will not properly clear 320x256 Layer 2, If you are in 320x256 mode 
you should use:

	CLS320(n)

> **NOTE**: This command has been deprcated and replaced with the more flexible [ClearLayer2(nColour)](CLEARLAYER2.md )
