# CLIPTILE

## Syntax

```
ClipTile(x1, x2, y1, y2)
```

## Description

Sets the visible clip area of Tilemap layer. 

**Example**

Clip Tilempa window to maximum visible size :

```
ClipTile(0,255,0,255)
```

**Remarks**

This command is a shortcut configuring [CLIP_TILEMAP_NR_1B](CLIP_TILEMAP_NR_1B) 4 times.
