# ClipLayer2(x1, x2, y1, y2)

## Syntax

```
ClipLayer2(x1, x2, y1, y2)
```

## Description

Sets the visible clip area of Layer2.

**Example**

Clip Layer2 to maximum visible size :

```
ClipLayer2(0,255,0,255)
```
**Remarks**

This command is a shortcut configuring [CLIP_LAYER2_NR_18](CLIP_LAYER2_NR_18) 4 times. 

Requires: 

```
#INCLUDE <nextlib.bas>
```
