# INITSFX

## Syntax

```
InitSFX(8k bank)
```

## Description

Initialises the `AYFX` sound effect bank, you must set up interrupts using [InitMusic()](InitSFX.md)

See [InitMusic()](InitMusic.md) for more details.

#INCLUDE <nextlib.bas>
#INCLUDE <nextlib_ints.bas>
