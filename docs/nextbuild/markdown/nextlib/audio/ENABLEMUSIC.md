# ENABLEMUSIC

## Syntax

```
EnableMusic
```

## Description

Macro that starts music playback. You must have initialised the music engine using [InitMusic()](InitMusic.md)

## Requires:
```
#DEFINE IM2 
#INCLUDE <nextlib.bas>
#INCLUDE <nextlib_ints.bas>  
```
