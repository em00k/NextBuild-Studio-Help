# ENABLESFX

## Syntax

```
EnableSfx
```

## Description

Macro that starts `AYFX` playback. You must have initialised the `AYFX` engine using [InitSFX()](InitSFX.md)

See [InitMusic()](InitMusic.md) for more details.

## Requires:
```
#DEFINE IM2
#INCLUDE <nextlib.bas>
#INCLUDE <nextlib_ints.bas>  
```
