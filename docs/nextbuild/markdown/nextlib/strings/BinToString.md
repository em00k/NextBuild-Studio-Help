# BINTOSTRING(uByte)

## Syntax

```
 n$ = BinToString(uByte)        
```

```
Dim a as uByte = $F0
Print BinToString(uByte)       
```
Will print: 
```
11110000
```
## Description

Returns a string binary representation of a uByte.
