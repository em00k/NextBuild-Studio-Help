# NextRegA(register,variable)

## Syntax

```
NextRegA(register,variable)
```

## Description

Sets Next register with value. 

```
a = 38
NextRegA(MMU1_2000_NR_51,a) : 'Sets reg $51 memory slot 1, $2000-$3fff to value in a, 38. 
```
