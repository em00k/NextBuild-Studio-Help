---
name: 'TileMap(address,offset,nrtiles,x,y,width,worldwidth)'
prefix: TileMap
description: >-
  Does a software L2 256x192 tilemap. address = mapdata, offset = block to start
  counting from, nrtiles = how many tiles to draw, x/y = x/y position to top
  left corner of tilemap, width = width of tilemap on scree, worldwidth = nr of
  tiles x in world
---

```
TileMap(${1:address},${2:0},${3:nrtiles},${4:x},${5:y},${6:width},${6:worldwidth})
```
