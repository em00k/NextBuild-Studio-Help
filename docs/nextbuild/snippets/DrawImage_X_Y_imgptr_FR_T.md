---
name: 'DrawImage(X, Y, *Image, frame, transparent)'
prefix: DrawImage
description: |-
  Draw an image at x,y on Layer2 256x192.

  *Image is a pointer to the image data.
  frame is the frame to draw.
  transparent is the transparent colour.
---

```
DrawImage(${1:x},${2:y},${3:*Image},${4:frame},${5:transparent})
```
